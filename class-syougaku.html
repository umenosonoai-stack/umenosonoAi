01Courses — 3つのクラス

大脳の完成期に、
Coreな刺激を。

小学校期は、大脳の発達が完成期を迎える大切な時間。処理のスピードではなく「考える力」そのものを育てます。

Core Learning

Coreラーニング

具体物を使って算数を学ぶ、処理ではなく考える算数。積み木で速さ・割合・濃度などを目に見える形で理解します。

編成 能力別定員 8名程度
Core Thinking

Coreシンキング

図形力の育成に特化。試行錯誤を繰り返すことで、気づき・発想の転換・イメージ力を養います。低学年は直感思考、高学年は論理思考へ。

編成 能力別定員 10名程度
IQ Print

知能プリント

ギルフォード知能構造論にもとづく教材で、150の知能因子を万遍なく刺激。図形・記号・概念の思考転換を訓練します。

編成 能力別定員 8名程度

02Results — 実績

算数オリンピック
ファイナルで、満点。

CUBEの塾生は、算数オリンピックのファイナルで満点を達成しています。「考える算数」の積み重ねは、確かな結果につながります。
